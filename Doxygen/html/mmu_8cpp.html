Detailed Description

Routines to translate virtual addresses to physical addresses.

Software sets up a table of legal translations. We look up in the table on every memory reference to find the true physical memory location.

One single types of translation are supported here, a linear page table. The virtual page # is used as an index into the table, to find the physical page #.

NB: one could also use a TLB in replacement or addition of the linear page table (not integrated in the current version of the source code).
