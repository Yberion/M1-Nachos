Detailed Description

Constructor & Destructor Documentation

◆ Timer()

Timer::Timer ( VoidFunctionPtr  timerHandler,
int  callArg,
bool  doRandom 
)

Initialize a hardware timer device. Save the place to call on each interrupt, and then arrange for the timer to start generating interrupts.

Parameters
timerHandleris the interrupt handler for the timer device. It is called with interrupts disabled every time the the timer expires.
callArgis the parameter to be passed to the interrupt handler.
doRandomif true, arrange for the interrupts to occur at random, instead of fixed, intervals.
