Detailed Description

This class is used to simulate an Asynchronous Communicating Interface Adapter on top of Unix sockets.

The system dependent ACIA provides emissions and receptions of bytes using sockets. An emission and a reception can be done in parallel (full duplex).

void ACIA_sysdep::InterruptRec ( )
