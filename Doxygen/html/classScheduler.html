◆ SwitchTo()

void Scheduler::SwitchTo ( Thread *  nextThread)

Causes a context switch to nextThread.

Dispatch the CPU to nextThread. Save the state of the old thread, and load the state of the new thread.

Note: we assume the state of the previously running thread has already been changed from running to blocked or ready (depending). Side effect: The global variable g_current_thread becomes nextThread.

Parameters
nextThreadis the thread to be put into the CPU.
