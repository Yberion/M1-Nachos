Detailed Description

Defines the "condition variable" synchronization tool.

A condition variable does not have a value, but threads may be queued, waiting on the variable. These are only operations on a condition variable:

Wait() – relinquish the CPU until signaled,

Signal() – wake up a thread, if there are any waiting on the condition

Broadcast() – wake up all threads waiting on the condition
