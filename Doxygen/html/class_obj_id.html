Detailed Description

Definition of object identifiers.

The ObjId class defines a set of object identifiers. By object, we mean every Nachos object (Process, Thread, Semaphore, Lock; etc). The class stores the list of created objects and for each of them associates an object identifier than can be passed to subsequent system calls on the object.

A method allows to detect of an object corresponding to a given identifier exists; this is used to check the parameters of system calls.
